## Overview

An eye-catching and modern developer portfolio showcasing creativity, performance and seamless user experience, built with Next.js, React, GSAP and Tailwind CSS.

## Built With

- Next.js
- React.js
- JavaScript
- Tailwind CSS
- GSAP

## Run Locally

```bash
npm install
npm run dev
```

Open:

```
http://localhost:3000
```
